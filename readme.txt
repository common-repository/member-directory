== Description ==

In backend, admin can enable / disable specific user roles shown in user member directory.
In backend, admin can bulk remove user from member directory by user id in Roles panel.

In backend, admin can set Nav Bar font size in Member Directory Settings panel
In backend, admin can set member directory term font size in Member Directory Settings panel
Opt to hide count number of member directory items in Member Directory Settings panel
Opt to significant display of digital superscripts on Navigation Bar  in Member Directory Settings panel

Shortcode:
insert [ member_directory ] shortcode in any wordpress post, we will display list all members on your site in  a memebr directory

[ member_directory role='teacher' ] shortcode supports using the "role" parameter to display users with a specific role in the member directory
(You can select users roles in member directory user roles settings panel too)

[ member_directory  include=â€™653,5,864â€™] shortcode supports using the "include" parameter to display users with a specific user ID list in the member directory

Improved function of member directory navigation bar 

= Member Directory for WordPress Version 2.0.5 =
In Roles panel, admin can bulk remove user from member directory by user id

= Member Directory for WordPress Version 2.0.1 =
New option to set up user roles of member directory
new menu item Member Directory Settings, you can:
opt to custom Nav Bar font size
opt to custom member directory term font size
opt to hide count number of member directoryitems
opt to significant display of digital superscripts on navigation bar 

= Member Directory for WordPress Version 1.3.1 =
In wordpress member directory list, display user's "Biographical Info" filed (user description)

= Member Directory Version 1.3.0 =
Added "Member Directory" menu item
If you click "Member Directory" menu item, you will open "Member Directory Global Settings" panel
In "Member Directory Global Settings" panel, you will find "Enable / Disable Specific User Roles Show in User Member Directory" option box
In Enable / Disable Specific User Roles Show in User Member Directory  option box, 
you can select multiple user roles from user roles lists and select to enable or disable specific user roles listed in member directory 

= Member Directory Version 1.2.0 =
Added navigation bar at the top of the members directory page. 
By default, we will list all users in the members directory page, in navigation bar, you can click A to view all users start A, click T to view all users start T
Also in the navigation bar, we will show user counts in the digital superscripts of the 'A', 'B', 'C'... and so on

== Installation ==

1:Upload the Member Directory plugin to your blog
2:Activate it 
3:Inser shortcode [ member_directory ] in any page or post
1, 2, 3: You're done!
